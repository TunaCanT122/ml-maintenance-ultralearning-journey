# ml-maintenance-ultralearning-journey
Documenting my progress through a 10-module machine learning curriculum to build a predictive maintenance pipeline

I chose to work on the AI4I 2020 Predictive Maintenance Dataset for my first major AI ultralearning project.

Looking at the current landscape of the Taiwanese tech sector, companies are looking for AI skills. Since I am more likely to get into an equipment-related field, I wanted a project that sat perfectly in the middle, a Smart Manufacturing Pipeline that bridges the gap between hardware and machine learning.
